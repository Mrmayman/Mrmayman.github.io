Windows x86_64 (Recommended)
Not sure which one you have? Click → About This Mac
NOTE: On macOS, if you have a "corrupted", "damaged" or "unverified" error, run this in the terminal: 
xattr -rc /Applications/quantum-launcher.app
Linux aarch64 (Asahi/Snapdragon/Raspberry Pi) Linux armv7h 32-bit (old Raspberry Pi)
Or install it as a flatpak 
flatpak remote-add quantum-launcher oci+https://mrmayman.github.io/ql-flatpak-remote/
flatpak install io.github.Mrmayman.QuantumLauncher
Or as a Fedora COPR 
sudo dnf copr enable apicalshark/quantum-launcher-rpm
sudo dnf in quantum-launcher
