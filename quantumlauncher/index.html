Quantum Launcher

By Mrmayman

A simple, fast Minecraft launcher

  • Lightweight, clean and incredibly fast.
  • Install forge/fabric/optifine/quilt with a click!
  • Built-in mod store and mod preset system
  • Play old versions seamlessly with Omniarchive
    integration and skin/sound fixes
  • Fully free and open source!
  • Now in version 0.3.1!
